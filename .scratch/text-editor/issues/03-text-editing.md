# 03: 文本编辑

Status: ready-for-agent

## Parent

[PRD: 全屏彩色文本编辑器](../PRD.md)

## What to build

在 Issue 02 的光标导航基础上，加入文本编辑能力：插入字符、删除字符、换行。这是编辑器核心交互。

核心行为：
1. 可打印字符（ASCII 0x20-0x7E，以及多字节 UTF-8 字符）在光标位置插入，光标右移
2. Backspace（0x08 或 0x7F）删除光标前一个字符，光标和后续内容左移
3. Enter（0x0D）在光标处插入换行符 `\n`，光标移到新行首
4. 每次编辑操作后增量更新行偏移索引数组
5. 任何编辑操作后置位 dirty flag（修改标记），触发顶栏显示 `[*]`
6. 总缓冲区大小不超过 32KB——插入时若超出则拒绝操作（可蜂鸣或忽略）

## Acceptance criteria

- [ ] 在光标位置输入字符，字符正确插入，后续内容后移，光标右移
- [ ] 在行首按 Backspace，当前行内容合并到上一行末尾（如非第一行）
- [ ] 在行中间按 Backspace，删除光标前字符，后续内容左移
- [ ] 按 Enter，光标处插入换行，原光标后内容移到新行
- [ ] 编辑操作后顶栏显示 `[文件名]*`（修改标记出现）
- [ ] 插入字符导致总大小超过 32KB 时拒绝操作，不崩溃
- [ ] 在文件开头 Backspace 不导致崩溃
- [ ] 在文件末尾 Enter 不导致崩溃
- [ ] 编辑后行号正确更新（新增行或删除行后行号连续）
