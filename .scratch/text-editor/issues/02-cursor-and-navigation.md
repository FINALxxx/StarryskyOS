# 02: 光标与导航

Status: ready-for-agent

## Parent

[PRD: 全屏彩色文本编辑器](../PRD.md)

## What to build

在 Issue 01 的静态渲染基础上，加入光标跟踪和方向键导航。编辑器不再"按任意键退出"，而是进入交互循环，通过方向键在文本中移动光标。

核心行为：
1. 逐键读取输入，识别方向键 ANSI 转义序列（`↑`=`ESC[A`, `↓`=`ESC[B`, `→`=`ESC[C`, `←`=`ESC[D`）
2. 维护光标逻辑位置（行号、列号），限制在有效文本范围内（不越界）
3. 渲染时在编辑区对应位置显示/高亮光标（ANSI 反色或颜色反转）
4. 底栏实时更新为 `Ln X, Col Y | Ctrl+W 保存  Esc 退出`
5. 方向键以外的按键暂不处理（留待 Issue 03）

## Acceptance criteria

- [ ] 打开文件后，光标初始位于第 1 行第 1 列（文件开头）
- [ ] 按 `↓` 光标移到下一行，到达最后一行后不再下移
- [ ] 按 `↑` 光标移到上一行，到达第一行后不再上移
- [ ] 按 `→` 光标右移一列，到达行尾后不再右移（或跳到下一行首）
- [ ] 按 `←` 光标左移一列，到达行首后不再左移（或跳到上一行尾）
- [ ] 底栏行列号随光标移动实时更新
- [ ] 空文件（只有一行空内容）时方向键不导致崩溃
- [ ] 渲染帧率在 115200 波特率下可用
